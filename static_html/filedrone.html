File Drone Procedures

  1. Importing Series Results
    1. As series results arrive, save the zip files to the import folder inside your league folder.
    2. Use the Transfer - Import Statistics command in DMB to import the results
    3. If you are unsure what to do, see DMB's Help information under "Transfers"
    4. Autoplay any unplayed Series
  2. Update the Web Site
    1. Go to the File Drone Page
    2. Enter the new series number and the deadline
    3. Click on "Update Series Info"
    4. If there any tie bids for free agents, enter the ranking of the tied teams (the team with the worst record gets preference)
  3. Update the DMB File
    1. Go to The Trades page on the web site and enter any trades in DMB
    2. Go to The Released Players page on the web site and enter the releases in DMB
    3. Go to The Free Agent Claims page on the web site and enter the claims in DMB (you should farm these players)
  4. Exporting and Sending the Series File
    1. In DMB, use the Transfer -Export League Database command to create the next series file
    2. Go to The Team Page on the web site and click on "$$league_name$$" link (this will open an e-mail with all the owners e-mail addresses)
    3. Write some witty and pithy comments about the series just played, also be sure to state what the current series number is and when the results must be returned
    4. Attach the file exported from DMB, it should be in the export folder inside your league folder
    5. Send the e-mail (this is very important - do not skip this step)
